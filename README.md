# Amazon_Fresh_Analytics


Project Overview :
Amazon Fresh Analytics is a SQL-based data analysis project designed to solve real-world business problems in the e-commerce and retail domain. The project focuses on managing and analyzing customer, product, supplier, inventory, order, and review data to generate meaningful business insights and support data-driven decision-making.


This project aims to design a relational database and perform SQL-based analysis to:
Optimize operational efficiency
Improve customer satisfaction
Monitor product demand and inventory
Analyze customer purchasing behavior
Generate revenue insights for better business decisions

Skills Gained
Data Modeling and ER Diagrams
SQL Queries and Aggregations
Database Management (DDL & DML)
Joins and Subqueries
Database Normalization
ACID Transactions and TCL Concepts
Business Problem Solving using SQL

Tools & Technologies Used
MySQL
SQL
Excel

Key Insights
Identified top selling product categories contributing highest to revenue
Analyzed customer purchasing behavior in different cities
Detected inventory demand trends for better stock management
Generated actionable insights from customer reviews and ratings.
